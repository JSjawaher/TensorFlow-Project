# TensorFlow AI Project 🤖

## 📌 وصف المشروع
مشروع ذكاء اصطناعي تم تطويره باستخدام مكتبة TensorFlow عبر Google Colab، ويهدف إلى تطبيق مفاهيم التعلم العميق وبناء نماذج تعلم آلي بسيطة لفهم كيفية تدريب النماذج ومعالجة البيانات وتحسين الأداء.

---

## 🧠 التقنيات المستخدمة
- Python  
- TensorFlow  
- Google Colab  
- NumPy  
- Matplotlib  

---

## 🎯 أهداف المشروع
- تطبيق مفاهيم التعلم العميق (Deep Learning)
- بناء نموذج ذكاء اصطناعي باستخدام TensorFlow
- فهم خطوات تدريب النماذج وتحسين الأداء
- التعامل مع البيانات وتجهيزها للنماذج
